# MindCoach API Dokümantasyonu

Bu klasör, MindCoach API için interaktif HTML dokümantasyonunu içerir.

## Dosyalar

- `index.html` - Ana dokümantasyon sayfası
- `styles.css` - Stil dosyası
- `script.js` - JavaScript fonksiyonları

## Kullanım

Dokümantasyonu görüntülemek için `index.html` dosyasını bir web tarayıcısında açın:

```bash
# macOS
open docs/index.html

# Linux
xdg-open docs/index.html

# Windows
start docs/index.html
```

Veya bir web sunucusu ile çalıştırın:

```bash
# Python ile
cd docs
python -m http.server 8000

# Node.js ile (http-server gerekli)
npx http-server docs -p 8000
```

Sonra tarayıcınızda `http://localhost:8000` adresine gidin.

## Özellikler

- ✅ Responsive tasarım (mobil uyumlu)
