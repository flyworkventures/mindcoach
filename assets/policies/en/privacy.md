MINDCOACH – Privacy Policy

Last Updated: 2025

This Privacy Policy explains the principles governing the collection, use, storage, and protection of personal data belonging to users of the MindCoach mobile application (“Application”). By using the Application, you are deemed to have accepted this Policy.

MindCoach prioritizes user privacy and data security as fundamental principles. All personal data is processed in compliance with applicable data protection regulations (KVKK, GDPR, and other relevant laws).

1. Information We Collect

The Application may collect the following types of data to provide its services:

1.1. Information Provided by the User

Name, nickname, or profile information

Email address (for registration, purchases, and support)

Messages and content created within the Application

User preferences, selected character traits, and personalization settings

Feedback, requests, and support messages

1.2. Automatically Collected Information

Device type, operating system, model, and technical details

Approximate location data (city/country level)

Application usage patterns (session duration, screen interactions, etc.)

Performance logs, error reports, and crash data

1.3. Third-Party Integrations

Authentication services (Google, Apple, etc.)

Analytics, performance, and app store measurement tools

Collected data is processed solely to operate, improve, and enhance the user experience within the Application.

2. Purposes of Data Use

Personal data may be used for the following purposes:

Ensuring the proper and uninterrupted functioning of the Application

Personalizing the chat experience

Remembering user preferences and providing improved interaction

Error detection, performance analysis, and optimization

Responding to user inquiries and support requests

Enhancing AI models through anonymized data

Fulfilling legal obligations

User conversations and personal data are never sold to third parties.

3. Data Retention Period

Personal data is retained based on the following criteria:

As long as the user’s account remains active

For the duration required by legal obligations

Support-related records may be stored for up to 12 months after resolution

When a user deletes their account, all personal data is permanently and irreversibly deleted from the system.

4. Data Sharing

Data may be shared only under the following circumstances:

Legal requirements: Court orders and official authority requests

Service providers: Cloud infrastructure, analytics, debugging, and payment systems

User consent: Data shared explicitly with user permission

Shared data is anonymized wherever possible, and the principle of minimum data use is observed.

5. Cookies and Tracking Technologies

MindCoach may use cookies and similar technologies to measure performance and improve user experience. These technologies are used for:

Service continuity

Performance optimization

Remembering user preferences

These technologies do not serve personal tracking or profiling purposes.

6. User Rights

Under KVKK and GDPR, you have the following rights:

Right to access your personal data

Right to request correction

Right to request deletion (“right to be forgotten”)

Right to restrict data processing

Right to data portability

Right to withdraw consent

To exercise these rights:
📩 support@fly-work.com

7. Security Measures

MindCoach implements technical and administrative measures to protect user data, including:

SSL/TLS encryption

Firewalls and access controls

Authorized access protocols

Regular security audits

While no system can be guaranteed to be entirely secure, MindCoach aims to maintain high security standards.

8. Children’s Privacy

MindCoach is not intended for individuals under the age of 13 and does not knowingly collect data from this age group. If such use is detected, the account will be terminated, and related data will be deleted.

9. Changes to This Policy

MindCoach may update this Privacy Policy from time to time. Changes become effective once published within the Application, and users will be notified accordingly.
